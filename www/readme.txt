Projeto para fazer soma de dois valores

Desenvolvedor: Tayná Alves Rodrigues
Contato: taynaalro@gmail.com (11) 984662926

Para rodar o projeto é necessário fazer um git clone do link: https://github.com/tutyrodrigues/calc-taqe.
Abrir a pasta WWW no seu editor de códigos e iniciar o IONIC SERVE no seu terminal.

O app possui um menu que da acesso há duas telas, uma onde possui os dois campos a serem preenchidos com os valores, assim que feito, a soma aparece automáticamente para o usuário. 
A outra tela possui uma breve apresentação da pessoa que desenvolveu o projeto.

Foi utilizado html, css, npm, ionic 1, angularJS e nodeJS no projeto.
